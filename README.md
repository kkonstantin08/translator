# LinguaPop AI 🌍✨

**LinguaPop AI** is a next-generation Chromium extension for seamless English ↔ Russian translation and writing assistance, powered by the **Mistral AI** language model. 

It moves beyond traditional machine translation by understanding context, preserving markdown formatting, natively rewriting your text, and replacing DOM elements on-the-fly without breaking the layout.

## ✨ Key Features

* **📖 Smart Dictionary (Single Word):** Highlight any word to instantly see a rich popup with alternatives, part of speech, pronunciation (IPA), and contextual examples.
* **📝 Contextual Phrase Translation:** Highlight sentences or paragraphs. The model translates them with high accuracy, preserving the original tone and Markdown formatting.
* **✍️ Writing Assistant (Reverse Translation):** Click the floating LinguaPop button inside any `<textarea>` or input field. Write in your native language, and the AI will rewrite it into perfect, native-sounding English (or vice versa), instantly replacing the text right in the input box!
* **🌊 Waterfall Page Translation:** Translate entire web pages with a beautiful top-to-bottom streaming effect. The extension intelligently batches paragraphs and replaces them live as responses arrive, bypassing long waiting times.
* **⚡️ Lightning-Fast LRU Cache:** 100% local `chrome.storage` caching mechanism. Repeated phrases or words load instantly (0ms) and save your API tokens.
* **🔊 Native Text-to-Speech:** Listen to the pronunciation of translated text directly in the popup.
* **🎨 Beautiful UI/UX:** Crafted with modern web standards, featuring Dark/Light mode support, fluid animations, and a non-intrusive Shadow DOM popup system.

## 🛠️ Tech Stack

* **Architecture:** Chrome Extension Manifest V3
* **Frontend:** React, TypeScript, Vite
* **Content Injection:** Vanilla TS + Shadow DOM (to completely isolate CSS)
* **LLM Integration:** Direct API calls to Mistral (configurable via settings)
* **Storage:** `chrome.storage.local`

## 🚀 Installation & Development

### 1. Build the Extension
Ensure you have Node.js installed.
```bash
npm install
npm run build
```

### 2. Load into Chrome / Edge / Brave
1. Open your browser and navigate to `chrome://extensions`
2. Enable **Developer mode** (usually a toggle in the top right corner)
3. Click **Load unpacked**
4. Select the `dist` directory generated by the build step.

## ⌨️ How to Use

1. **Setup:** Click the extension icon in your toolbar, go to Settings ⚙️, and paste your **Mistral API Key**.
2. **Read:** Highlight text on any website. A tiny LinguaPop action button will appear. Click it (or press `Alt+T`) to see the translation.
3. **Write:** Focus any text input on the web (like a comment box). Click the pink LinguaPop button in the corner to magically rewrite your draft.
4. **Full Page:** Right-click anywhere on the page and select **Перевести страницу на русский (LinguaPop)**.

### Keyboard Shortcuts
| Shortcut | Action |
| --- | --- |
| `Alt + T` | Translate the currently selected text (or full page if nothing is selected) |
| `Ctrl/Cmd + Shift + L` | Open the manual popup translator in the toolbar |
| `Escape` | Close any active inline popups |

## 🔒 Privacy & Security

**Your data stays yours.** 
LinguaPop only sends the text you actively choose to translate (or the page content when explicitly requested) directly to the Mistral API. 
* API Keys are stored locally and securely in your browser's `chrome.storage`.
* The translation cache is strictly local and uses an LRU (Least Recently Used) algorithm to prevent memory bloat.
