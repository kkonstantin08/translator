# LinguaPop AI — Design System

## Visual Direction

Minimalistic, polished, SaaS-like interface with a warm, playful accent.

## Colors

```css
--color-canvas-white: #ffffff;
--color-midnight-plum: #240029;
--color-whisper-gray: #d4ccd4;
--color-deep-plum-text: #333333;
--color-rose-bloom: #df37a7;
--color-muted-orchid: #6d526d;
--color-sunshine-gold: #ffcc11;
--color-warning-red: #ef4444;
```

## Typography

- Primary: Inter / system-ui / sans-serif
- Clean, readable sizes
- No paid or unavailable fonts

## Components

- White cards with 14px border-radius
- Subtle purple-tinted shadows
- Compact polished buttons
- Primary buttons in `#df37a7`
- Pill badges for alternatives/tags
- Clean input borders with strong focus states
- Compact beautiful popup near selected text
- Small unobtrusive floating selection button
